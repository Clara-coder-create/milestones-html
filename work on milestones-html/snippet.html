The a tag has for attributes: href, id, class, target. The values are: href="https://google.com" id="the-best-link" class="banana" target="_blank"

href: creates the hyperlink for the anchor tag id: gives the tag a unique identity in the html document and makes it easier to link it to css or a java skript. https://developer.mozilla.org/en-US/docs/Web/HTML/Global_attributes/id class: helps select certain elements in the html skript, similarly to the id. However, the class is not a unique attibute. The same class can be used for different tags. target: specifies how the link will be opened. the default is in the current window. if target = _blank the link will open in a separate tab.
